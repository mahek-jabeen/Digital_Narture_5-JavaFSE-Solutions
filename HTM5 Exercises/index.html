Console Panel

Press F12 → Console tab. You'll see all console.log(), console.info(), console.warn(), and console.error() outputs from this page in real time.

Elements Panel

Inspect any element by right-clicking → "Inspect". You can live-edit the DOM, modify CSS properties, and see how changes affect layout immediately.

Network Panel

Switch to the Network tab and reload the page to see all HTTP requests — HTML, CSS, fonts, images, and video — with timing and size data.

Application Panel

Go to Application → Storage → localStorage to see the event preference you saved in Exercise 8. You can view, edit, and delete keys here.
