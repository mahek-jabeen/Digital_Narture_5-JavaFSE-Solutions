Exercise 19 – Sass Customization

Customizing Bootstrap with Sass

What is Sass?

Sass (Syntactically Awesome Style Sheets) is a CSS preprocessor that extends CSS with variables, nesting, mixins, and functions. Bootstrap is written entirely in Sass, giving developers full customization control.

Overriding Bootstrap Variables

Import your custom variables before Bootstrap's own variable file. This ensures your overrides take precedence throughout the entire framework via the cascade.

// custom.scss
$primary: #e91e63;
$border-radius: 1rem;
@import "bootstrap";
Color & Border Customization

Change the primary brand color and global border radius in one place. Every component (buttons, cards, alerts, inputs) automatically inherits the new values — no individual overrides needed.

$primary: #6200ea;
$border-radius: 0.5rem;
$font-size-base: 1rem;
