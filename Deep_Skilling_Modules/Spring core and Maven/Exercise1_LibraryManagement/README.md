# Exercise 1 – Configuring a Basic Spring Application

## Project Name
Exercise1_LibraryManagement

## Objective
Demonstrate:
- Spring Core
- XML Configuration
- Bean Definition
- Dependency Injection
- Application Context

## Technology
- Java 17
- Maven
- Spring Framework 6

## How to Run
1. Compile the project:
   `mvn compile`
2. Run the application:
   `mvn exec:java -Dexec.mainClass=com.library.MainApp`
