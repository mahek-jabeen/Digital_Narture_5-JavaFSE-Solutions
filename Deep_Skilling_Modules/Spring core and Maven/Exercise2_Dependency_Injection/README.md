# Exercise 2 – Implementing Dependency Injection

## Project Name
Exercise2_Dependency_Injection

## Objective
Demonstrate:
- Spring IoC
- Dependency Injection
- Setter Injection
- XML Bean Configuration

## Technology
- Java 17
- Maven
- Spring Framework 6

## How to Run
1. Compile the project:
   `mvn compile`
2. Run the application:
   `mvn exec:java`
